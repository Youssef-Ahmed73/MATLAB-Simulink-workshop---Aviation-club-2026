# MATLAB & Simulink Workshop - Aviation Club 2026

A complete MATLAB and Simulink workshop developed and delivered for members of the Aviation Club in 2026.



## Workshop Structure

| Session | Topic                                      |
| ------- | ------------------------------------------ |
| 1       | Introduction to MATLAB                     |
| 2       | Programming Basics                         |
| 3       | Matrices and Functions                     |
| 4       | Data Visualization                         |
| 5       | Curve fitting and signals processing       |
| 6       | Introduction to Simulink                   |
| 7       | Modeling and Simulation with Simulink      |
| 8       | Arduino Integration with MATLAB & Simulink |
